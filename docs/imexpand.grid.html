Convert locations of im3d voxel grid into XYZ coordinates

Usage

imexpand.grid(d)

Arguments

d
An im3d object

Value

Nx3 matrix of image coordindates

Description

Convert locations of im3d voxel grid into XYZ coordinates

Examples

d=im3d(,dim=c(2,3,2),origin=c(10,20,30),voxdims=c(1,2,3))
imexpand.grid(d)
