Write object to VTK file

Usage

write.vtk(x, file, ...)

# S3 method for neuron
write.vtk(x, file, datatype = c("float", "double"),
  title = file, WriteAllSubTrees = TRUE, ...)

Arguments

x
Object to write
file
Path to output file
...
Additional arguments to methods
datatype
The VTK data type (one of float or double)
title
Title of the .vtk file (defaults to file)
WriteAllSubTrees
Whether to write all subtrees in the neuron or just the main tree.
